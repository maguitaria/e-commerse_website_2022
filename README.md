### E- commerse website
We created a website with sections for man, women and kids. All pages are responsive, include change of currency and shopping cart.

## Here is deployment [link](https://maguitaria.github.io/e-commerse_website_2022/)

## This is a final project of HTML and CSS programming course.
  # Functionalities:
  * plain HTML, CSS, JavaScript and PHP
  * website with 10 responsive, animated pages
  * currency change
  * responsive deisgn, fancy animation
  * form to stay in touch with clients
